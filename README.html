3 Project

The purpose of this application is produce several pdf files from an xlsx file, as a re-implementation of https://github.com/IronBrushTattoo/cj as a web application.

3.1 User Story

  • user logs in
  • chooses xlsx file for upload
  • selects number of days back to make labels from
  • submits
